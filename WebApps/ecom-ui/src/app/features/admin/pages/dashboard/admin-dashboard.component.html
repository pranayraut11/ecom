Getting Started

This admin interface allows you to manage your store's visual appearance and layout configuration:

  • Themes: Define color schemes for your store
  • Components: Configure individual UI elements like headers, footers, etc.
  • Layouts: Arrange components into complete page layouts

All changes are reflected immediately on your store front-end.
